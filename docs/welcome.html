Quick Start Guide

Follow these 4 simple steps to start exporting your highlights

1

Create a Notion Integration

Go to Notion's integrations page and create a new integration. Copy the "Internal Integration Token" - you'll need this!

Open Notion Integrations
2

Set Up Your Database

Create or select a Notion database with at least "Title" and "Author" properties. Share it with your integration via the database's Share menu.

Database ID is in the URL: notion.so/DATABASE_ID?v=...
3

Configure the Extension

Click the extension icon in your browser toolbar, go to Settings, and enter:

  • Notion API Token
  • Database ID
  • Title Property Name
  • Author Property Name
  • Your Kindle Region
4

Export Your Highlights!

Navigate to your Kindle highlights page, click the extension icon, and hit "Export to Notion". That's it!
